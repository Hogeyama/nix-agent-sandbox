:::note[名前について]
当初は Nix 統合を主目的として *nix-agent-sandbox* と命名しました。その後、ネットワーク制御・コマンド移譲・Worktree 管理など様々な機能が追加された結果、Nix 統合は数ある機能のひとつに過ぎなくなっています。この機能が不要な場合は、実行環境に Nix がインストールされている必要もありません。
:::
